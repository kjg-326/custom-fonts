# Food Producer Company

## Overview
This project is a simple website for a food producer company. It includes a homepage, an about page, and a shop page, showcasing the company's products and providing information about its mission and values.

## Project Structure
```
food-producer-company
├── index.html       # Homepage with hero section, product display, and navigation
├── about.html       # Detailed information about the company
├── shop.html        # Product listings with descriptions and pricing
├── css
│   └── style.css    # Styles for the website
└── README.md        # Project documentation
```

## Features
- Standout hero section on the homepage
- Call to action for users to explore products
- Brief information about the company
- Display of products available for purchase
- Navigation buttons to the about page and shop page

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the homepage.
3. Navigate to the `about.html` and `shop.html` pages using the provided navigation buttons.
